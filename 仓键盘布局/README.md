# 仓输入法键盘布局文件使用

针对 iOS 端的仓输入法，做了个键盘布局，放在了文件夹 `/仓键盘布局` 中

```sh
ios-仓输入法键盘布局-2026-05-15.yaml
```

这个键盘布局跟 iOS 原生键盘布局十分类似。


## 使用方法：
1. 打开仓输入法的设置
2. 找到 **自定义键盘**，点击右上角的 + 号，选择 **导入配色与布局**，然后选择上面那个文件 `ios-仓输入法键盘布局-2026-05-15`
3. 导入成功
4. 再回到最外层菜单，选择 **RIME**，点击里面的 **重新部署** 按钮就可以了
